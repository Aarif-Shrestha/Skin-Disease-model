# Derma Vision  
Skin Disease Classification using Deep Learning (CNN)

##  Overview
Derma Vision is a deep learning-based model designed to classify skin disease images using a Convolutional Neural Network (CNN).  
The model takes an input image and predicts the likely category of skin condition based on learned patterns from the training dataset.

>  This repository contains only the trained model.
